# **App Name**: WanderPersona

## Core Features:

- Interactive Quiz: Guide users through a step-by-step quiz with a clean, modern UI.
- AI Persona Generation: Utilize an LLM tool to generate a unique travel persona, description, location, and image prompt based on quiz answers.
- Result Display: Display the persona title, AI-generated image, description, and recommended destination on a dedicated screen.
- Viral Shareability: Enable users to share their persona on social media with a generated shareable image that includes the persona title, AI art, and a call-to-action watermark.

## Style Guidelines:

- Primary color: Dusty rose (#C4A4A4), providing warmth and sophistication.
- Background color: Off-white (#F5F5F5), providing a soft, clean backdrop to emphasize content.
- Accent color: Muted mauve (#B784A7), complementing the primary color with a hint of creativity and wanderlust.
- Headline font: 'Playfair' serif for an elegant, fashionable feel.
- Body font: 'PT Sans' sans-serif complementing Playfair with modernity and readability. 
- Minimalist icons, complementing the quiz's personality questions.
- Use a centered layout for the quiz, with a fixed width to maintain focus, and animate the container height as a user proceeds.
